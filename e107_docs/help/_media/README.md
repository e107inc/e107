# e107 admin help – shared media

Drop screenshots / figures here and reference them from any help topic with the
`SHOT>` marker, for example:

    SHOT> users/new-user-form.png | Creating a new user

Paths are resolved relative to this folder (`e107_docs/help/_media/`).

For external images or images outside this folder, use `IMG>` with either a
full URL or an absolute path.
